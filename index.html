  • API_KEY_INVALID → ตรวจ Vercel env → GEMINI_API_KEY
    • 
    
  • PERMISSION_DENIED → เปิด Gemini API ใน Google Cloud Console

  • 404 / model not found → model name ผิด (แก้แล้วใน ocr.js)

  • CORS error → domain ยังไม่ใช่ vercel.app (ทดสอบที่นี่ได้เลย)
    • 
    
  • fetch failed → ocr.js ยังไม่ได้ push ขึ้น GitHub
